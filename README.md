# Scientific Analysis 2026

## Abstract

This project presents a statistical analysis of student performance data using Python.

The research investigates whether there is a statistically significant difference in mathematics scores between male and female students.

The study follows Open Science principles and provides reproducible code, datasets, and documentation.

---

# Dataset

Dataset used:

Students Performance in Exams

Source:
https://www.kaggle.com/datasets/spscientist/students-performance-in-exams

---

# Methodology

The following methods were used:

- Data cleaning
- Descriptive statistics
- Data visualization
- Independent t-test
- Comparative analysis

Libraries used:

- pandas
- matplotlib
- scipy

---

# Installation

Clone repository:

```bash
git clone https://github.com/USERNAME/scientific-analysis-2026.git
cd scientific-analysis-2026
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# Run Project

Run Jupyter Notebook:

```bash
jupyter notebook notebooks/lab4_analysis.ipynb
```

or run Python script:

```bash
python src/analysis.py
```

---

# Results

The statistical analysis showed that:

- there is a significant difference in math scores between genders;
- preparation courses positively affect student performance.

Visualizations and statistical outputs are available in the notebook.
